# 掌心窗 v0.3.2 · 页面重构版

本版重点不是新增大功能，而是重新整理四个页面的职责：

- 总览：生活仪表盘，只放老公提醒、电量、当前 App、屏幕时间、当前地区天气和生活细节。
- 看见：保留日常入口「给老公看一眼」，测试截图文案退场。
- 守护：应用门禁、天气地区、主动提醒、周期提醒、回家模式集中管理。
- 设置：主题、连接、权限、应用包名和高级调试工具。

调整内容：

1. 大量测试按钮从日常页面移入设置页的调试抽屉。
2. 「测试截图上传」改为「给老公看一眼」。
3. 背景减粉，默认主题更偏奶油绿/薄雾白。
4. 按钮和抽屉入口继续缩小，弱化工程后台感。
5. 底部导航保持四项，但高度更轻。
