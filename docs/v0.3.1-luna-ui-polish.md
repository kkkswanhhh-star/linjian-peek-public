# 掌心窗 v0.3.2 · 页面重构 UI polish

本版不新增大功能，专门打磨界面：

- 真主题切换：奶油绿、雾蓝白、白桃粉、夜航黑、薄荷透明会即时影响背景、卡片、按钮、底部导航与文字颜色。
- 字体和字重调整：标题更圆润，正文更轻，按钮不再厚重。
- 卡片轻量化：淡边框、轻阴影、大圆角，减少“工程后台”感。
- 按钮缩小：主题按钮、抽屉入口、动作按钮统一变成小 chip / 细条。
- 抽屉入口重做：长内容用「标题 ›」形式收纳，展开后再显示详情。
- 底部导航更轻：保留总览 / 看见 / 守护 / 设置，选中项为轻胶囊。
- 首页继续保留生活总览：老公提醒、电量、当前 App、屏幕时间、当前天气。

目标：掌心窗从“调试后台”继续往“柔软、干净、日常可用的生活小窗”靠近。
