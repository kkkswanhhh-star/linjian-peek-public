# v0.3.4-public-clean

公开版同步自用版 v0.3.4 的页面结构和主要功能，保留“老公”关系语感，同时清理私人信息。

## 同步内容

- 总览 / 看见 / 守护 / 设置四页 UI。
- 生活细节默认折叠。
- 看见页小卡片和小按钮。
- 应用门禁抽屉。
- 天气地区抽屉与真实天气状态。
- 主动提醒和周期提醒抽屉。
- MCP 工具补全。

## 公开版处理

- 默认设备 ID：`android-phone`。
- 默认称呼：对方 `老公`，自己 `宝宝`，可在设置里修改。
- 默认地区使用通用示例。
- 包名白名单增加 Gemini、Claude、微博、X。
- 不内置私人姓名、私人城市、私人账号或私人口令。
