# v0.3.0 生活总览版：天气地区 + 简约 UI

这版把掌心窗从“工程调试台”改成更适合日常打开的生活小窗。

## 核心变化

1. **总览页重做**
   - 首页只显示电量、当前 App、今日屏幕时间、当前天气地区和“老公提醒”。
   - 包名、日志、测试按钮不再糊在第一屏。

2. **底部导航精简**
   - 从 6 个入口改为 4 个：总览 / 看见 / 守护 / 设置。
   - “控制、状态、门禁、日志”的工程式入口被整理到守护和设置里。

3. **天气地区**
   - 支持保存多个地区，默认：家 / 学校 / 常去地。
   - 只能有一个“当前地区”。
   - 掌心窗生活状态会返回 current_weather_location 和 weather_locations，MCP 可基于当前地区查天气。

4. **天气提醒**
   - App 端会把当前地区和天气备注上传到生活状态。
   - MCP 新增 get_weather_state，可按当前地区查实时天气并生成出门建议。
   - MCP 新增 send_weather_notification，可把天气提醒发到手机通知。

5. **抽屉收纳**
   - 连接设置、权限运行、本机测试、应用包名、回家模式、主动提醒、生理期、调试日志都改成抽屉。
   - 长内容默认收起，需要排查时再展开。

6. **主题入口**
   - 新增主题设置入口：奶油绿、雾蓝白、白桃粉、夜航黑、薄荷透明。
   - 当前版本先保存主题偏好并调整入口，后续可继续细化每套主题的背景、卡片和按钮颜色。

## MCP 新增工具

- `get_weather_state`
  - 读取手机生活状态中的当前天气地区。
  - 通过 Open-Meteo 查询实时天气。
  - 返回温度、高低温、降雨概率和出门建议。

- `send_weather_notification`
  - 查询当前天气地区。
  - 生成“老公口吻”的出门提醒。
  - 通过掌心窗通知发到手机。

## 注意

- 手机端保存地区与当前地区；实时天气查询由 MCP 侧完成。
- App 不固定早晚自动播报天气，适合“老公查掌心窗时顺手提醒”的用法。
- 天气通知可由 MCP 在需要时发出。
